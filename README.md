# 📈 Enterprise Sales Forecasting & Business Intelligence Platform

An end-to-end Machine Learning application that predicts future sales using historical sales data. The platform includes data validation, preprocessing, feature engineering, model training, forecasting, explainable AI, business insights, and interactive dashboards.

---

## 🚀 Features

### 📂 Data Management
- Upload custom sales dataset
- Use default dataset
- Automatic dataset validation
- Dataset summary and statistics

### 🧹 Data Preprocessing
- Date conversion
- Duplicate removal
- Sorting by date
- Index reset

### ⚙️ Feature Engineering
- Date-based features
- One-Hot Encoding for categorical variables
- Automatic feature preparation

### 🤖 Machine Learning
- Linear Regression
- Random Forest
- Gradient Boosting
- Extra Trees
- Auto Model Selection

### 📊 Exploratory Data Analysis
- Sales Trend
- Store Performance
- Correlation Heatmap
- Feature Distributions

### 📈 Forecasting
- Future sales prediction
- Interactive forecast charts
- Forecast table
- CSV download

### 💡 Business Intelligence
- Store performance analysis
- Holiday sales impact
- Weekly sales summary
- Business recommendations

### 🔍 Explainable AI
- Feature Importance
- Model Interpretation

### 📥 Download Center
- Evaluation Report
- Forecast CSV
- Model Comparison Report

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Sales-Forecasting-System.git
```

Move into the project

```bash
cd Sales-Forecasting-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
